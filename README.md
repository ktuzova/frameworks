# frameworks


 laboratory work on applied systems and artificial intelligence frameworks


Лабораторные работы по дисциплине "	Прикладные системы и фреймворки искусственного интеллекта"

Этот репозиторий содержит результаты выполнения 5 лабораторных работ по изучению и применению основных алгоритмов машинного обучения. Каждая работа включает в себя реализацию и исследование моделей для задач классификации и регрессии, а также сравнение их эффективности.


Содержание работ
Лабораторная работа 1: K-ближайших соседей (KNN)
Выбор уникальных наборов данных для классификации и регрессии
Создание и оценка базовой модели
Улучшение модели через препроцессинг и подбор гиперпараметров
Самостоятельная реализация алгоритма KNN

Лабораторная работа 2: Линейная и логистическая регрессия
Исследование линейной регрессии для задачи регрессии
Исследование логистической регрессии для задачи классификации
Сравнение с базовыми и улучшенными моделями

Лабораторная работа 3: Решающие деревья
Построение и оценка деревьев для классификации и регрессии
Анализ важности признаков и визуализация деревьев

Лабораторная работа 4: Случайный лес
Ансамблирование деревьев для повышения точности
Исследование влияния количества деревьев и глубины

Лабораторная работа 5: Градиентный бустинг
Исследование бустинговых алгоритмов (CatBoost, XGBoost, LightGBM)
Сравнение всех изученных алгоритмов по итоговым метрикам
